【1】احترامك لجميع المواطنين

【2】يمنع سحب ( مواطن أو اداري ) بدون استأذان

【3】يمنع الانتقال لاي مسؤول الا في حال ( طلب الاذن منه وموافقته ) أو يكتب بشات الاداره في اللعبه ( الي يبي مسؤول حياه )

【4】يمنع الانتقال لاي مواطن الا في حال ( شكوى ) عليه

【5】يمنع الانعاش في المناطق الامنه الا في حال ( طلب ممسعف وانتظار دقيقه كامله ) أو ( عدم وجود مسعفين )

【6】يمنع الانعاش في المناطق الغير امنه منعا باتا

【7】يمنع التدخل بأمور الوزارات نهائيا ( بل التوجه لرفع شكوى في الديسكورد )

【8】يمنع طلب الترقيات أو الاعتراض على ترقيات اشخاص اخريين

【9】يمنع الانتقال للعساكر أو المسعفين الا في حال ( شكوى تثبت ذلك )

【10】ممنوع اعطاء رتب لاي شخص دون التأكد من امتلاكها في الديسكورد

【11】 يمنع سجن المواطنين بدون ذكر السبب و المده شفوياً

【12】عقوبه الادمن أكثر من عقوبه المواطن ( لانك واجهه السيرفر )

【13】ممنوع ذكر كلمه ( هاك أو اسم ادمن ) في الشات العام باللعبه
يوجد شات الاداره باللعبه ( ADM/ ) شات الاداره

【14】ممنوع محاسبه ادمن ثاني أو التدخل بشؤونه ( تحفظ تصويرك وتروح لمسؤول )

【15】ممنوع تكون مسجل دخولك كأدمن وتلعب بالرول بلاي ( عسكري أو مسعف أو مواطن أو عصابه )

【16】ممنوع التدخل بمشكله قدامك أو تحاسب بدون تكت

【17】يمنع التكلم بأمور التهديد ( سجن أو الباند ) للمواطنين

【18】يمنع استخدام خواص الاداره أمام المواطنين

【19】جميع المخالفات للمواطنين لا تتعدى ( 20 شهرا )

【20】 يجب أن يكون اسمك باللعبه مطابق لاسمك في الديسكورد ( بدون زخرفه )

【21】ممنوع التدخل بأمور ( الهيتمان ) أو محاسبتهم يوجد مسؤولين

【22】في حال اي شخص بالسيرفر بدأ بـ ( السب أو القذف ) تحفظ تصويرك وتروح لمسؤول

【23】ممنوع استخدام ( الثغرات أو القلتشات ) في السيرفر

【24】 في حال كان المواطن اسلوبه كويس وتفهم غلطه مسموح لك بتخفيف عقوبته للنصف

【25】ممنوع سحب الاداريين في السبورت الا بأذن أو استخدام صلاحيات الديسكورد ( الميوت أو الدفن )

【26】ممنوع تسجل دخولك كأدمن وما تقبل تكتات

【27】يجب على كل اداري موجود بالسيرفر يكون مفعل القيم اكتفتي ومتواجد برومات الديسكورد

【28】ممنوع الانعاش داخل مقر العصابات الا ( اعضاء العصابه نفسها )

【29】ممنوع الانتقال بأي شخص بالسيرفر ( تعطيه سياره أو تطلب أدمن رتب )

【30】يمنع التكلم أو نشر سيرفرات اخرى داخل السيرفر

【31】ممنوع تلعب سيناريو وتكون مسجل دخولك كأدمن

تـعـريـفـات

ROLEPLAY : هي طريقه لتقمص شخصيه وهميه داخل اللعبه

VDM : هي استخدام المركبه كسلاح ( الصدم العشوائي )

RDM : هي الطلق بالمناطق الامنه ( القتل العشوائي )

في حال وجود طلب ادمن في المهجوره يمنع حل المشكله داخل المكان [ 32 ]

اي إداري يعرف ثغرة او قلتش ومايتواصل مع الاداره سيتم محاسبته وفي حال استخدامه سيتم فصله من الإدارة مباشرة [ 33 ]

يمنع منعا باتا التكلم بالسوء عن السيرفرات الاخرى او التطرق لـ اشخاص خارج السيرفر منعا باتا [ 34 ]

في حال كان المواطن محترم و متقبل للعقوبة يحق للادمن تخفيف السجن عليه الى نص المده [ 35 ]

يمنع سحب الإداريين في الرومات الصوتية بدون أذن او استخدام صلاحياتك بالديسكورد بشكل خاطئ [ 36 ]

عند تقديم استقالة واردت الرجوع الى الاداره لن يتم ارجاعك بنفس الرتبه [ 37 ]

في حال اللعب ك ادمن داخل السيرفر يجب تواجدك داخل رومات الديسكورد [ 38 ]

يمنع طلب اجازه اكثر من 3 ايام بدون سبب مقنع [ 39 ]

ممنوع انعاش اي رجل عصابه داخل مقره في وقت السيناريو حتى لو انتهى السيناريو [ 40 ]

يمنع منعا باتا توصيل المواطنين الا بحاله عدم وجود سائق اوبر [ 41 ]

في حال دخولك لسيرفر اللوقات يجب تغيير اسمك هناك ليتطابق مع اسمك هنا [ 42 ]

يجب استخدام شات الاداره داخل القيم في حال طلب مساعده من الاداريين الاخرين [ 43 ]

يمنع منع باتا انتقال الاداري الى الاداره العليا من غير طلب انتقال او اذن ب شات الاداره داخل اللعبه [ 44 ]
